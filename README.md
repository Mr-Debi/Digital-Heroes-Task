# 🛠 Utility Tools

### 1. EMI Calculator
- Calculate monthly EMI
- Loan amount
- Interest rate
- Tenure calculation

### 2. GST Calculator
- Calculate GST amount
- GST inclusive/exclusive values

### 3. QR Code Generator
- Generate QR codes
- Download QR code as PNG image

### 4. Password Generator
- Generate secure passwords
- Randomized characters

### 5. Resume Builder
- Multiple Resume Templates
  - Modern Professional
  - Software Developer
  - Fresher
- Live Resume Preview
- Download Resume PDF

### 6. PDF Merger
- Merge multiple PDF files
- Download merged PDF

### 7. Age Calculator
- Calculate age in:
  - Years
  - Months
  - Days
- Total days lived

### 8. JSON Formatter
- Validate JSON
- Pretty format JSON

### 9. Currency Converter
- Real-time exchange rates
- 160+ currencies
- Live conversion
- Exchange rate display

# ⚙️ Installation

## Clone Repository

```bash
git clone https://github.com/Mr-Debi/digital-heroes-portfolio.git
```

```bash
cd digital-heroes-portfolio
```

---

# 📦 Install All Dependencies

## Single Command

```bash
npm install react-router-dom qrcode.react jspdf pdf-lib axios react-icons react-dropzone file-saver html2canvas react-hook-form uuid
```

OR

```bash
npm install
```

if package.json already exists.

---

# 🚀 Run Project

```bash
npm run dev
```

Application will run at:

```text
http://localhost:5173
```

---

# 📦 Build Project

```bash
npm run build
```

---

# 👀 Preview Production Build

```bash
npm run preview
```

---

# 🌐 Deploy to GitHub Pages

Install gh-pages:

```bash
npm install gh-pages --save-dev
```

Deploy:

```bash
npm run deploy
```
